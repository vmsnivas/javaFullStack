# HTML

- Used to create a structure of a web page

## Elements

- Latest version : HTML5
    - <!doctype html>
- <> Open Tag
- </> Close Tag
- html - Root Element
- head - Info about the document
- body - Output
- title - Name of the doc
- p - Paragraph
- h1 - h6 : Headers

---

- Lists
    - ul : Unordered List - Bullet Format
        - li 
    - ol : Ordered List - Numbered List Format
        - li
    - dl : Description List
        - dt : Description Term
        - dd : Definition of the Description List

- table : Table
    - thead : Header row of the table
        - tr : Row
        - th : Cells in the Header Row
    - tbody : Body of the table
        - tr : Row
        - td : Cells in the data row

---

- Attributes
    - name="value"

- Style Attribute
    - Add info to element
    - Open tag
    - name="value"
    - Every element
    - Property
        - Customization of an element
        - name:value / key:value
        - Width
        - Height
        - background-color
        - color
        - text-align

        - Ex: <button style="width:150px; height:">

    - div
        - Part of webpage
