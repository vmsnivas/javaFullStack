What is?

HTML
HTML, or HyperText Markup Language, is the standard, foundational language used to create and structure web pages. It uses a system of tags and elements to define content types like headings, paragraphs, images, and links, telling web browsers how to display them. It is not a programming language, but a markup language that acts as the building block for nearly all websites.
CSS
CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation, styling, and layout of a document written in a markup language such as HTML. It works alongside HTML, which provides the structure and content of a web page, by defining how elements should appear on screens, in print, or on various devices.
JavaScript
JavaScript is a versatile, high-level programming language that adds interactivity and dynamic behavior to websites. It is one of the core technologies of the World Wide Web, working alongside HTML (which structures content) and CSS (which styles content) to create rich user experiences.
jQuery
jQuery is a fast, small, and feature-rich JavaScript library designed to simplify front-end web development. Its core philosophy is "Write less, do more" by wrapping many common, complex JavaScript tasks into concise methods.
Bootstrap
Bootstrap is a widely-used, open-source front-end framework for developing responsive, mobile-first websites. It provides a collection of pre-built CSS, HTML, and JavaScript components—such as grids, buttons, forms, and navigation bars—that simplify web design and speed up development. It is now primarily used for building fast and consistent layouts.
React JS
React.js (also known as React or ReactJS) is an open-source JavaScript library for building user interfaces (UIs). It is maintained by Meta (formerly Facebook) and a large community of individual developers and companies.
Redux
Redux is an open-source, predictable state management library for JavaScript applications, commonly used with React to centralize application data. It enables consistent behavior across environments, simplifies debugging with tools like time-travel debugging, and uses a single, immutable store to manage application state.
